### 4. Firestore Security Rules

Replace the default security rules in your Firestore settings with the following to secure your database:

```
service cloud.firestore {
  match /databases/{database}/documents {
    // Users can only read/write their own data
    match /users/{userId} {
      allow read, write: if request.auth.uid == userId;
    }

    // Allow authenticated users to read timetable and activities
    match /timetable/{docId} {
      allow read: if request.auth != null;
    }

    match /activities/{docId} {
      allow read: if request.auth != null;
    }
  }
}
```
